Welcome to Expnsr!

Expnsr is a simple to use checkbook register and budget planning application. It is built using the expnsr-api written as a RESTful API in Laravel 5.3. Expnsr's API allows users to create an account where they can create checkbook registers for as many accounts as they desire all for free! Expnsr also features a bill reminder service and a budget planning service.

Expnsr is a cross platform service built upon a RESTful API using Laravel 5.3, dingo-api and jwt-auth for Laravel. The plan is to expand the Expnsr service to Android and iOS devices.
